Tables

Tables have (almost) no default styles, in order to account for varying table designs. Note that tables are not layout modules, and must be wrapped in a mod in order to get the correct spacing applied if used as stand-alone block modules.

Simple tables with vertical lines

The class table enables a full-width table with some spacing and subtle vertical lines.

To control the width of individual rows, use the layout column classes, .c1 through .c12, plus the half-width .c1of24. In order to force one column to get all available space and give all other columns just the minimum available size, add .c12 to it.

Discrete tables

The table-discrete class yields a full-width table with no other styling. It is useful to simply tabularize data in the flow of text. Nest paragraphs inside for more spacing.

Expando table

The expando-table class creates an inverted table (e.g. dark on light background and the other way around) that can have highlighted rows and expanded rows.

For places where space is a scarse resource (like the search results), add the expando-tight class to make the table display with less white-space.
