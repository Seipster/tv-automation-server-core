Notifications

Notification count indicator. Typically used as a badge on an icon, but can also be standalone, for instance on menu items.

When used as a badge on an icon, wrap the .notification element in a .notifications element. When the icon is large (icon-l) use .notifications-l, and .notifications-xl for extra large icons.
