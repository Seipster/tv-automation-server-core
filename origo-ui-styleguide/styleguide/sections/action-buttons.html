Action buttons

Actions buttons are links or buttons that appear as text (without chrome). Action buttons are created with the action-btn class name:

If the action is composed of several items, apply (or wrap them with) the action-item class for alignment and spacing between.

Action items are vertically aligned to top, but if you need a baseline alignment instead, apply the action-vb class to the action.

Action buttons can be disabled, which will disable hover effect, and dim color.
