Media elements

The media element is a concept from OOCSS. It is a generic structure that can be used to solve surprisingly many component-level layouts. It has a fixed-width "thing" (typically a picture, but can be anything) on the left and free-floating content on the right, which does not flow below the left-side content even if it is longer. Read more about the media block.
