Grid flex layout

Page flex layout is controlled by two main classes, flex-row and flex-col.
The flex-row has additional classes for vertical alignment:
.vertical-align-top .vertical-align-bottom .vertical-align-middle .vertical-align-stretch .vertical-align-baseline
And horizontal alignment:
.horizontal-align-left .horizontal-align-right .horizontal-align-center .horizontal-align-space-between .horizontal-space-around The flex-col has additional classes for vertical alignment:
.vertical-align-top .vertical-align-bottom .vertical-align-middle .vertical-space-between .vertical-space-around
And horizontal alignment:
.horizontal-align-left .horizontal-align-right .horizontal-align-center
To control the column width, use:
.c1, c2, c3 ... .c12

There is also a class .c-1of24 for making half-width columns (useful for button-only columns etc).
