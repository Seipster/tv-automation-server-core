Buttons

Buttons come in two varieties: btn-primary and btn-secondary. Use with the btn class.

Buttons can also be made more compact by applying the btn-compact class. This can be useful for buttons that only display an icon.

All buttons can be made full-width by applying the btn-big class.

 

 

Tight/small buttons

Buttons also come in a tighter version. Use the btn-tight class with either btn-primary or btn-secondary.

 

 

Buttons with icons

Apply the .btn-icon-left class to properly align an icon placed to the left of the text.

Similarly, use the .btn-icon-right class for buttons with icons placed to the right.
