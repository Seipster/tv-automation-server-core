Responsive grids

Responsive grids are made from a bunch of boxes that stack from left to right in varying column counts on different screen sizes. These boxes default to 12em width. How many columns this creates depends on the width of the parent element. Alternative sizes will be added as needed. To create variable-sized inline-blocks, use the class grid-box.
